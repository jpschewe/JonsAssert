Jon's Assertion Package Download (@APP-VERSION@)

You need jdk-1.2.x or better to run/compile this. You'll also need the following jar files. I've listed the primary sites to give proper credit to the authors that wrote these great packages. You can see the major changes here. For a full list of changes see ChangeLog.

  • JonsAssert-@APP-VERSION@.jar
  • JonsInfra-0.2.jar Primary Site
  • werken.opt.jar Primary site. LICENSE
  • antlr.jarThis is version 2.7.1, I haven't tested it with 2.7.2x yet.Primary site
  • junit.jar if you want to build and run the testcases from the jar file. This is junit-3.8. Earlier versions should work, but I'd encourage you to upgrade to the newest version because of some bugs.Primary site

If you happen to use Ant then you'll be interested in this build.xml file. It's got an example of how to tie JonsAssert into an ant build.

Usage

	java -classpath "JonsAssert-@APP-VERSION@.jar:JonsInfra-0.2.jar:werken.opt.jar:antlr.jar" net.mtu.eggplant.dbc.JonsAssert [options] files ...

See the API documentation for JonsAssert for commandline options
